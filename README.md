<h1 id="vclinic">VClinic</h1>
<p>VClinic is a hospital app created by Divinity - The Code Lord ( Divine Chiedozie ) to help people find help to health issues.</p>
<h2 id="features">Features</h2>
<ul>
<li>The prescription tab of the app helps the users find help to simple health issues themselves.</li>
<li>The app also has a booking tab that helps users connect to various doctors around the world to help them.</li>
<li>They have the ability to either call the health practitioner on his line or book online.</li>
</ul>
