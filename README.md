<h2 id="features">Features</h2>
<ul>
<li>Material design</li>
<li>Browse Songs, Albums, Artists</li>
<li>Create and edit playlists</li>
<li>6 different now playing styles</li>
<li>Homescreen widgets</li>
<li>Browse device folders</li>
<li>Dark theme and UI customisability</li>
<li>Gestures for track switching</li>
<li>LastFM scrobble</li>
<li>Android Wear and Android Auto support</li>
<li>Playing queue in notification (Xposed)</li>
<li>Lyrics support</li>
<li>Chromecast support</li>
</ul>
